About VibeCode Blog

關於本站

VibeCode Blog 是一個基於 Node.js 和 jQuery 構建的現代化部落格系統，我們提供簡潔優雅的界面，讓您可以輕鬆地創建、編輯和管理文章，同時保持專注於內容本身。

本站支持 Markdown 編輯，讓您可以輕鬆地格式化文本、添加代碼塊、插入圖片等，並提供實時預覽功能。我們還提供標籤系統，幫助您組織和分類文章，讓讀者更容易找到感興趣的內容。

Node.js Express jQuery Markdown Responsive Design

主要功能

Markdown 編輯器
使用直觀的 Markdown 編輯器創建美觀的文章，支持代碼上色和實時預覽。
標籤系統
使用標籤對文章進行分類，輕鬆管理和搜索相關內容。
收藏功能
將喜歡的文章加入收藏，方便日後閱讀和分享。
搜索功能
快速搜索文章標題、內容和標籤，輕鬆找到所需資訊。
用戶認證
安全的用戶註冊和登入系統，保護您的內容和隱私。
響應式設計
完美適應桌面和移動設備，隨時隨地訪問和管理您的部落格。

技術堆疊

本站使用以下技術開發：

  • 前端：HTML5, CSS3, JavaScript, jQuery
  • 後端：Node.js, Express.js
  • 編輯器：SimpleMDE (Markdown 編輯器)
  • 代碼高亮：Highlight.js
  • 安全：DOMPurify (防止 XSS 攻擊)
  • 數據存儲：檔案系統 (JSON)
